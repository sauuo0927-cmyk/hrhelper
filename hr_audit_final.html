稽核規則說明
本系統依序執行三項檢核，任一不符即標記為加班異常
1比對表定與實際加班時段
取明細表 K欄（加班開始時間，如「17:30」），與出勤加班時數明細表 G欄（加班起訖時間）的前五個字元進行字串比對。

K欄前5碼 ≠ G欄前5碼 → 異常

例如 K欄為「17:30」，G欄為「17:30-19:30」，取前5碼均為「17:30」→ 符合。
2檢查多報或少報加班
以下公式結果若不為 0，代表申報與核准時數不一致：

出勤加班時數明細表.P欄 − 出勤加班時數明細表.Q欄 + 明細表.S欄 ≠ 0 → 異常

差額為正：多報；差額為負：少報。
3確認公假換休時數
僅當 明細表.S欄 為 0 或空值時才執行此項檢核。

先計算換休合計：
換休合計 = 公假換休時數表.I欄 + J欄 + K欄
再驗證：
出勤加班時數明細表.Q欄 − 換休合計 ≠ 0 → 異常

若未上傳第三份表，此項檢核自動跳過。
⚠ 跨月申報說明
本系統目前設定：若某筆明細表記錄在出勤加班時數明細表中找不到對應的姓名+日期， 則跳過不計入稽核（不視為異常）。

這是因為實務上員工可能在月底加班、下個月才申報（跨月申報），兩份表的匯出區間未必完全一致。
